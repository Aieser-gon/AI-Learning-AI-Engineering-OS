# AI-Learning-AI-Engineering-OS

Transformando aprendizado em evidências de engenharia. 

## Objetivo

Este projeto documenta um sistema construído para otimizar a organização do conhecimento utilizando IA, acelerando os estudos enquanto demonstra competências práticas de AI Engineering na minha jornada durante a Formação em
Artificial Intelligence Engineering da Indicium AI Academy e o Ciclo Preparatório do Lighthouse.

Mais do que registrar estudos, este repositório reúne projetos, experimentos, anotações e ferramentas desenvolvidas ao longo da formação.
